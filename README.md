# 📊 Student Marks Analysis

A Data Analysis project built using Python, NumPy, Pandas, and Matplotlib. This project analyzes student marks, calculates percentages, generates grades, rankings, and visual reports.

---

🚀 Features

- Read student data from CSV file
- Calculate Total Marks
- Calculate Percentage
- Generate Pass/Fail Result
- Assign Grades Automatically
- Find Topper Student
- Find Lowest Performing Student
- Generate Student Ranking
- Subject-wise Average Marks
- Subject-wise Highest Marks
- Subject-wise Lowest Marks
- Filter Students Above 80%
- Bar Chart Visualization
- Pie Chart Visualization
- Export Final Report to CSV

🛠 Technologies Used

- Python
- NumPy
- Pandas
- Matplotlib
- CSV File Handling
